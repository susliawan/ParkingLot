The parking lot program is created using programming language Java with ParkingLot.jar as executable file.
How to run the program:
1. Open terminal and go to folder that contains ParkingLot.jar
2. To run the program based on input command file, 
	type "java -jar ParkingLot.jar [file_input_path] [file_output_path]"
3. To run the program in interactive way based on user input,
	type "java -jar ParkingLot.jar"
4. To exit the program type "exit" in input command
